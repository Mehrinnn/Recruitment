# Git Task

## Task Description

The objective of this task was to complete the Git Mastery platform exercises at both the Beginner and Advanced difficulty levels. The task focused on understanding and practicing Git commands and workflows used in software development.

## Implementation

I completed the Git Mastery challenges by performing various Git operations, including branch management, merging, rebasing, commit handling, and repository navigation. These exercises helped me in improving my understanding of Git workflows.

## What I Learned

* Creating and managing branches
* Merging branches
* Rebasing commits
* Understanding commit history
* Navigating repositories using Git commands
* Working with common Git workflows
